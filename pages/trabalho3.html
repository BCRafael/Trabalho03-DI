Trabalho 3: APIs e JavaScript Assíncrono

Este trabalho mostra a interação com APIs usando fetch, Promise.all e tratamento de erros com try/catch.

O que foi implementado

  • Carregamento dos dados da prova a partir de um arquivo JSON local.
  • Uso da API de tradução MyMemory em um formulário.
  • Consulta simultânea a 3 APIs diferentes com Promise.all.
  • Tratamento de erros em todas as requisições.
  • Componentes reutilizáveis de cabeçalho, navegação e rodapé.
