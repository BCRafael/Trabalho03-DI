Tradução com API MyMemory

Informe um texto em português e veja a tradução para outro idioma usando a API MyMemory. A requisição é feita com fetch e o resultado aparece abaixo.

Formulário de Tradução

Digite um texto para traduzir.

Resposta da API

Preencha o formulário e clique em Traduzir.

Como funciona

O texto é enviado para a API MyMemory. Em seguida, o servidor retorna a tradução correspondente e essa resposta é exibida ao usuário.

Todas as requisições usam try/catch para tratar falhas de rede e erros de resposta.

Esta API não exige chave para o exemplo usado.
