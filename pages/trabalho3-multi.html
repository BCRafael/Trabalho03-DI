Consulta a 3 APIs Diferentes

Nesta página usamos Promise.all para buscar dados simultaneamente de três APIs diferentes e apresentar o resultado ao usuário.

Ghibli

Pressione o botão para ver um filme.

IBGE Notícias

Pressione o botão para ver a notícia mais recente.

Spaceflight News

Pressione o botão para ver a notícia espacial.

Como funciona

A página faz três chamadas de API em paralelo usando Promise.all. Se qualquer requisição falhar, mostramos uma mensagem explicativa ao usuário.

APIs usadas:

  • Ghibli Films
  • IBGE Notícias
  • Spaceflight News
